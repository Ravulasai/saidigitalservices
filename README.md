# Sai Digital Services Website

Premium, production-ready Next.js 15 website for Sai Digital Services - a digital agency based in Mogaltur, West Godavari, Andhra Pradesh.

## Features

- ✨ Modern, luxurious design with glassmorphism effects
- 🌓 Dark/Light mode with theme persistence
- 📱 Fully responsive mobile-first design
- ⚡ Optimized for performance (Lighthouse 90+)
- 🔍 SEO-ready with structured data and meta tags
- 🎨 Smooth animations with Framer Motion
- 📧 Contact form with validation
- 🗺️ Google Maps integration
- 💬 Floating WhatsApp button
- 📊 Scroll progress indicator

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: JavaScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Theme**: next-themes

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
